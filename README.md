# 酒店管理系统 java8

## 项目概述

酒店管理系统是一款基于Java和Vue的前后端分离系统，使用SpringBoot框架进行开发。该系统致力于为酒店行业提供高效、便捷的管理解决方案。

## 技术栈

### 系统架构
- B/S架构
- MVC模式

### 运行环境
- 系统环境：Windows/Mac

### 开发环境
- 开发工具：IDEA
- JDK版本：1.8
- 项目构建：Maven
- 数据库：Mysql

### 技术框架
- 后端：Java、SpringBoot、Mybatis
- 前端：Vue

## 功能模块

### 管理员模块
- 统计分析
- 会员管理
- 广告管理
- 分类管理
- 房间管理
- 订单管理
- 评价管理
- 预约管理

### 用户模块
- 注册登录
- 房间浏览查询
- 预约房间及评价
- 个人信息修改

## 系统角色
- 管理员
- 普通用户

## 目录结构

```
hotel_management_system/
├── backend/       # 后端代码目录
│   ├── src/
│   ├── pom.xml
│   └── ...
└── frontend/      # 前端代码目录
    ├── src/
    └── ...
```

## 核心亮点
- 前后端分离，便于维护与扩展
- 采用成熟技术栈，确保系统稳定性和可维护性
- 功能全面，覆盖酒店管理及用户操作主要需求

## 部署步骤

1. 准备运行环境：安装Java JDK 1.8、Mysql数据库、Maven
2. 导入数据库文件到Mysql
3. 使用IDEA等工具导入后端项目，构建并运行
4. 使用WebStorm等工具导入前端项目，构建并运行
